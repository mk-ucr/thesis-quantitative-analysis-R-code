# thesis-quantitative-analysis-R-code
Psychedelic-Assisted Therapy for PTSD: Analysis Code
This repository contains the R code for the quantitative analyses in my thesis on psychedelic-assisted therapy for posttraumatic stress disorder (PTSD), submitted to University College Roosevelt in 2026.
The thesis investigates how four substances (MDMA, psilocybin, LSD, and ketamine) influence PTSD through neurobiological, experiential, and clinical pathways. The code here reproduces the three figures presented in the Results section.
